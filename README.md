📊 Online Food Delivery Analysis
1.Project Overview
          This project analyzes an Online Food Delivery dataset to extract insights on customer behavior, delivery                        performance,and business trends using Python, SQL, and Power BI.
2.Objectives
       Understand customer ordering patterns
       Identify high-revenue cities and cuisines
       Analyze delivery efficiency
       Study cancellation reasons and discount impact
3.Dataset
      Records: 100,000
      Features: 25 columns (Customer, Order, Delivery, Restaurant details)
4.Data Preprocessing
      Handled missing values (Median / Mode)
      Converted data types (Date, Time)
      Feature Engineering:
      Weekday vs Weekend
      Peak Hour
      Age Groups
      Profit Margin %
5.Analysis Performed
      City-wise revenue analysis
      Cuisine-wise performance
      Peak hour demand analysis
      Distance vs delivery time
      Payment mode preferences
      Cancellation reason analysis
6.SQL Queries
      Total Orders & Revenue
      Top Customers
      Monthly Revenue Trends
     Cancellation Rate
     Delivery Performance
7.Power BI Dashboard
      KPI Metrics
	  Total Orders
      Total Revenue
      Average Order Value
      Average Delivery Time
      Cancellation Rate
      Average Rating
8.Tools & Technologies
       Python (Pandas, NumPy, Matplotlib, Seaborn)
       SQL (MySQL)
       Power BI
Conclusion:
This project demonstrates how data analysis helps improve business decisions, operational efficiency, and customer satisfaction.
